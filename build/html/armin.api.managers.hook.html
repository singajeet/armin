Register a method or function as hook point Args: namespace (str): Register an hook under given namespace. Any other callback registered under this namespace will be called before or after this function executes allow_call_type (F): If the value is PRE, only callbacks registered under this namespace for Pre-execution will be called. Other options are POST, BOTH
